氣泡排序 (Bubble Sort)

透過動畫觀察元素逐一比較與交換位置，讓氣泡排序的流程更直觀、易懂。

操作原理

氣泡排序是一種簡單的排序演算法，其基本思想是：

  1. 從序列的起始位置開始，比較相鄰的兩個元素
  2. 如果前一個元素大於後一個元素，則交換它們的位置
  3. 對每一對相鄰元素重複上述過程，直到序列的最後
  4. 重複整個過程，直到沒有需要交換的元素

這個過程就像氣泡一樣，較大的元素會逐漸"浮"到序列的末尾。

Python 代碼實現

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        # 標記是否發生交換
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                # 交換元素
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        
        # 如果沒有發生交換，說明已經排序完成
        if not swapped:
            break
    
    return arr

# 範例使用
arr = [64, 34, 25, 12, 22, 11, 90]
print("原始陣列:", arr)
bubble_sort(arr)
print("排序後:", arr)

複雜度分析

  • 時間複雜度：
    • 最壞情況：O(n²)
    • 最佳情況：O(n) - 當陣列已經排序時
    • 平均情況：O(n²)

    需要進行 n(n-1)/2 次比較，最壞情況下需要同樣次數的交換。

  • 空間複雜度：O(1)
  • 穩定性：穩定排序
