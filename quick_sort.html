快速排序 (Quick Sort)

選取基準值分割序列，遞迴排序左右區間，平均效能優秀。

操作原理

快速排序選取一個基準值（pivot），將序列分成小於基準值和大於基準值兩部分，然後對兩部分遞迴排序。

  1. 從序列中選取基準值
  2. 將小於基準值的元素移到左邊，大於基準值的元素移到右邊
  3. 對左邊和右邊的子序列各自遞迴執行快速排序
  4. 直到子序列長度為 1 或 0，排序完成

Python 代碼實現

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

arr = [3, 6, 8, 10, 1, 2, 1]
print(quick_sort(arr))

複雜度分析

  • 時間複雜度：平均 O(n log n)，最佳 O(n log n)，最壞 O(n²)
  • 空間複雜度：O(log n)
  • 穩定性：不穩定排序
  • 原地排序：一般實作為否
