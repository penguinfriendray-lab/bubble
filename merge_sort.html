合併排序 (Merge Sort)

採用分治策略，將序列遞迴切分後合併，時間複雜度穩定在 O(n log n)。

操作原理

合併排序透過遞迴將序列切分成較小的子序列，直到每個子序列長度為 1，接著將它們兩兩合併成有序序列。

  1. 將序列分成兩個子序列
  2. 遞迴對每個子序列執行合併排序
  3. 將兩個已排序子序列進行合併
  4. 重複以上步驟直到整個序列排序完成

Python 代碼實現

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    merged = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            merged.append(left[i])
            i += 1
        else:
            merged.append(right[j])
            j += 1
    merged.extend(left[i:])
    merged.extend(right[j:])
    return merged

arr = [38, 27, 43, 3, 9, 82, 10]
print(merge_sort(arr))

複雜度分析

  • 時間複雜度：平均 O(n log n)，最佳 O(n log n)，最壞 O(n log n)
  • 空間複雜度：O(n)
  • 穩定性：穩定排序
  • 原地排序：否
