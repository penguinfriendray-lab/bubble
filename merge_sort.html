合併排序法

Merge Sort

分治法的經典應用，將大問題分解為小問題後合併解決

合併排序法說明

合併排序法是一種分治法（Divide and Conquer）的排序演算法。它將陣列不斷地分成兩半，直到每個子陣列只剩一個元素，然後將這些已排序的子陣列合併起來。

運作原理

合併排序的核心思想是將一個大陣列分成兩個較小的子陣列，遞歸地對這兩個子陣列進行排序，然後將兩個已排序的子陣列合併成一個有序陣列。

程式碼範例

def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])
    return result

時間複雜度

  • 最佳情況： O(n log n)
  • 最壞情況： O(n log n)
  • 平均情況： O(n log n)

空間複雜度

O(n) - 需要額外的空間來儲存臨時陣列
