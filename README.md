# 排序演算法視覺化報告

## 📋 報告概述

這份報告探討了常見的排序演算法，包括其原理、實作方式、時間與空間複雜度分析，以及互動式視覺化展示。

## 🎯 學習目標

- 理解不同排序演算法的基本原理
- 分析各演算法的時間與空間複雜度
- 透過視覺化動畫觀察演算法的執行過程
- 比較不同演算法的優缺點

## 📚 報告內容

### 1. 氣泡排序法 (Bubble Sort)
- **原理說明**: 重複比較相鄰元素並交換位置
- **時間複雜度**: O(n²)
- **空間複雜度**: O(1)
- **優點**: 實作簡單，穩定排序
- **缺點**: 效率較低，不適合大型資料

### 2. 選擇排序法 (Selection Sort)
- **原理說明**: 每次找出最小元素與當前位置交換
- **時間複雜度**: O(n²)
- **空間複雜度**: O(1)
- **優點**: 交換次數少，穩定排序
- **缺點**: 不穩定，效率較低

### 3. 插入排序法 (Insertion Sort)
- **原理說明**: 將元素插入已排序序列的適當位置
- **時間複雜度**: O(n²)
- **空間複雜度**: O(1)
- **優點**: 穩定排序，對小資料集效率高
- **缺點**: 大資料集效率低
